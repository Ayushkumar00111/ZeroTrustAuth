# Task Management App

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB
- JWT Authentication
- TailwindCSS

## Features
- User authentication (JWT based)
- Protected routes
- Task CRUD
- Search and filter
- Password hashing (bcrypt)

## Scalability
Frontend and backend are separated.
JWT allows stateless authentication.
Project structured modularly for scaling.
